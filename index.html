What is WordPress?

  • WordPress started in 2003 as a blogging platform
  • Today it is used as a CMS.
  • Powers eCommerce,social sites,etc.
  • 40% of websites today are on WordPress according to builtwith.com

WordPress.com

  • Has ads
  • Easy to use
  • No domain name needed

WordPress.org

  • Learning curve
  • Fully customizable
  • More plugins and theme choices
